# üåå Obsidian Flow: LuminaLearn ‚Äî Design Language

## 1. Design Philosophy

* **Focused Immersion** ‚Üí The UI dissolves away, leaving only content (video, notes, progression).
* **Fluid Interactions** ‚Üí Every state change feels like water flowing: dissolves, ripples, soft morphs.
* **Encouraging Progression** ‚Üí Progress is always visible but never loud ‚Äî glowing bars, ripple checkmarks, subtle milestone celebrations.
* **Playful Sophistication** ‚Üí Dark, serious foundation with playful glowing accents (blue/violet/green) used sparingly for delight.

---

## 2. Color System (Refined)

### Core Palette

* **Background Deep**: `#05050C` ‚Üí near-absolute black (canvas).
* **Liquid Glass Base**: `#1A1A2E` @ 70% opacity + blur (main panel material).
* **Lumina Blue** (Primary): `#007AFF` ‚Üí electric glowing blue (actions, progress).
* **Success Green**: `#34C759` ‚Üí emerald glow (completion, achievements).
* **Insight Violet**: `#AF52DE` ‚Üí deep violet (secondary highlights, knowledge points).
* **Text Primary**: `#F0F0F0` (luminous white).
* **Text Secondary**: `#A0A0A0` (soft neutral grey).

### Glow & Shadow Rules

* **Active Glow**: outer glow (`0 0 12px Lumina Blue`) for active buttons/tabs.
* **Success Glow**: soft ripple outward (`0 0 10px Success Green`) on completion.
* **Violet Halo**: subtle inner-shadow with `Insight Violet` for optional/secondary cues.

---

## 3. Typography System

* **Headings**: Rounded geometric sans (Plus Jakarta Sans Rounded / Inter Rounded) ‚Äî luminous, friendly, but strong.
* **Body**: Inter / Work Sans ‚Äî clean, neutral.
* **Notes Editor**: Fira Code / JetBrains Mono ‚Äî monospaced, with subtle violet highlights for markdown.

**Scale (Dark UI optimized)**

* H1 ‚Üí 40px, glowing edge subtle
* H2 ‚Üí 32px
* H3 ‚Üí 24px
* Body ‚Üí 16px
* Caption ‚Üí 13px

Use **slight text glow (blue/violet)** for headings in active contexts (like course titles).

---

## 4. Layout System

* **Dashboard/Home** ‚Üí 2-column grid (main path cards left, quick links + milestones right).
* **Lesson Browser** ‚Üí Toggle list/gallery view with minimalist pill buttons.
* **Split Screen** ‚Üí 60/40 split (video/notes), draggable divider with glowing Lumina Blue hover.
* **Panels** ‚Üí All major UI in frosted liquid glass (`#1A1A2E70` + blur-md).

**Hierarchy**:

* Top nav minimal (icon-only sidebar or floating).
* Dashboard cards = big, glowing pathfinders.
* Lesson rows = sleek, immersive panes.

---

## 6. Animations & Micro-interactions

* **Ripple Press**: every button tap sends a faint liquid ripple across the surface.
* **Tab Switch**: dissolve/reform transition (opacity + blur + ripple wave).
* **Progress Fill**: smooth ‚Äúliquid pour‚Äù rather than linear bar fill.
* **Milestones**: shimmering wave + ripple burst around the icon.
* **Notes Cursor Glow**: Lumina Blue pulse follows cursor while typing.

**Motion Parameters**

* Ease: `cubic-bezier(.25,.8,.25,1)` (fluid).
* Durations:

  * Micro (hover, taps): 180‚Äì220ms
  * Mid (progress, cards): 400‚Äì600ms
  * Major (tab/page transitions): 600‚Äì800ms

---

## 7. Sidebar Navigation ‚Äî ‚ÄúLet‚Äôs Learn‚Äù

* Icons: book (Courses), beaker (Labs), trophy (Achievements).
* **Active Tab**: glowing Lumina Blue halo + animated underline ripple.
* **Inactive Tabs**: soft white-grey with faint hover glow.

Transition: tabs dissolve into liquid glass and reform when switching.

---

## 8. Accessibility & Clarity

* **Contrast**: Lumina Blue (#007AFF) on black (#05050C) has \~8.5:1 ratio ‚Üí excellent.
* **Success Green & Violet**: use with larger text/icons only; not for small body text.
* **Keyboard**: glowing outline ring (Lumina Blue) on focus.
* **Motion Prefs**: `prefers-reduced-motion` ‚Üí replace ripples with instant color/opacity changes.

---

## 9. Mood & Atmosphere

* **Background**: pure black with **subtle animated gradients** (Lumina Blue ‚Üí Violet wave at 5% opacity).
* **Overall Feel**: like studying in a futuristic obsidian-glass workspace, with glowing knowledge nodes lighting up as you progress.
